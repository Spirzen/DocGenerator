# DocGenerator
Данное приложение берёт Markdown-файлы +YAML-шаблоны и генерирует красивый PDF/HTML с оглавлением, подсветкой кода, темами.

# Интерфейс
Программа позволяет пользователю использовать интерфейс с кнопками:
Выбрать Markdown (файл с расширением .md);
Выбрать шаблон YAML (файл с расширением .yaml, содержащий метаданные - автора и версию, например);
Выбрать тему (выпадающий список, светлая или тёмная);
Выбрать формат (выходного файла - PDF или HTML);
Создать документ (кнопка для запуска генерации).
Выбрать место сохранения для файла;
Ввести текст для документа вручную и очистить поле ввода текста.

# Логика работы
# Шаг 1
Чтение файлов - программа читает содержимое выбранного Markdown-файла и затем читает YAML-файл с метаданными.

# Шаг 2
Парсинг YAML - Используется библиотека YamlDotNet для анализа YAML-файла и преобразования его в словарь (Dictionaty<string, object>).

# Шаг 3
Обработка Markdown - Библиотека Markdig преобразует Markdown в HTML с использованием расширенных возможностей (например, поддержка таблиц, списков и т.д.).

# Шаг 4
Загрузка HTML-шаблона - программа загружает HTML-шаблон из внедрённых ресурсов проекта (html_template.html)

# Шаг 5
Применение шаблонизатора Handlebars - библиотека используется для вставки данных в HTML-шаблон.

# Шаг 6
Генерация выходного файла - для PDF используется SelectPdf, а HTML сохраняется в файл.

# Шаг 7
Создаётся файл output.pdf или output.html в зависимости от выбранного формата.
